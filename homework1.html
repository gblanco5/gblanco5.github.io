Resources

  • HTML Dog - This website offers a variety of resources for new and old Web Developers. The website is organized in four parts: tutorials, techniques, examples, and references. The tutorials are helpful to new developers trying to learn the coding language or older ones trying to brush up on some old skills. Techniques explain the ways different coding languages can be used together to achieve a common goal. The examples section brings code to life and gives the user a visual representation of what they are trying to accomplish. Lastly the references section has a list of all the tags organized by sections.
  • HTML Color Codes - This website is completely focused on colors. While it may seem extreme to have it listed as an important resource, I've found finding the perfect shade of a certain color can be extremely difficult if you are limited by simple color names such as "red." It's much more convenient to use a color picker to find the ideal color and have the html code of said color provided for your convenience.
